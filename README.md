# ML-fundamentals-2026

## Overview
This project focuses on the end-to-end data preparation and feature engineering of the UCI Bank Marketing dataset. The primary goal is to transform raw marketing data into a high-quality format suitable for machine learning, specifically for predicting term deposit subscriptions.
